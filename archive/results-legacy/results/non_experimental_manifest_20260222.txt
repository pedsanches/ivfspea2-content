# Non-experimental artifact manifest (2026-02-22)

This manifest lists artifacts generated without running new optimization experiments.
Generation scripts are deterministic over existing archived files.

## Regeneration commands

1) Rebuild ablation from local archived `.mat` files:
   matlab -batch "run('scripts/rebuild_ablation_from_local_data.m')"

2) Compute M2 IGD-HV cross-check (IVF/SPEA2 vs SPEA2):
   matlab -batch "run('scripts/hv_m2_crosscheck.m')"

3) Integrate modern baselines into main synthetic benchmark from archived runs:
   matlab -batch "run('scripts/integrate_modern_baselines_main.m')"

4) Build runtime competitiveness summary vs modern baselines:
   matlab -batch "run('scripts/build_runtime_modern_summary.m')"

5) Regenerate main synthetic boxplots with modern baselines:
   matlab -batch "run('scripts/regenerate_main_boxplots_with_modern.m')"

6) Build detailed M2/M3 IGD tables with modern baselines and best-per-problem highlighting:
   matlab -batch "run('scripts/build_detailed_tables_with_modern.m')"
