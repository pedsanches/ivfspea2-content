# Archive

This directory holds historical, deprecated, or reviewer-irrelevant materials that
were intentionally moved out of the main repository surface without being deleted.

Current reviewer-facing entry points remain at the repository root, especially:

- `README.md`
- `paper/`
- `docs/IVFSPEA2_KNOWLEDGE_BASE.md`
- `docs/IVFSPEA2_EVIDENCE_MODEL.md`
- `results/SUBMISSION_EVIDENCE_MAP.md`
- `results/submission_release_manifest.csv`
- `CITATION.cff`
- `src/`
- `tests/`

Archive groups:

- `archive/editorial-history/` - review trackers, rebuttal notes, and internal review files
- `archive/docs-historical/` - planning notes, audits, and deprecated support documents
- `archive/docs-procedural/` - execution protocols, track plans, and workflow notes
- `archive/paper-workfiles/` - superseded manuscript PDFs and non-final figure variants
- `archive/reference-material/` - local reference PDFs and background material
- `archive/results-legacy/` - deprecated outputs, backups, and superseded result files
- `archive/results-procedural/` - execution logs, packaging notes, and superseded traceability docs
- `archive/data-legacy/` - backup processed data files no longer needed in the main surface
- `archive/scripts-legacy/` - legacy helper scripts retained for history
